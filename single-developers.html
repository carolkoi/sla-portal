  • Quick Summary

1.1 What is this repository for?

B2B Payment Processing Integration Manual

1.2 How do I get set up?

  • Ensure that you have a merchant account with switch link africa.
  • Ensure your account has been set up to allow transactions
  • Ensure that you have a security key

1.3 Contribution guidelines

We would really appreciate feedback and suggestions. Issues, and suggestions can be emailed to technical [at] switch link africa [dot] com

1.4 Who do I talk to incase of any query?

  • For general issues, email support [at] iswitch link africa [dot] com support@switchlink.com
  • For technical issues, email technical [at] switch link africa [dot] com technical@switchlink.com
  • Or send us a skype request at switch link africa support support@switchlink.com

1.5 URI & Versioning

We hope to improve the API over time. The changes won’t always be backward compatible, so we’re going to use versioning. There is currently no rate limit. For versioning purposes, only removal of a non-optional field or alteration of an existing field will be considered incompatible changes. You should gracefully handle additional fields you do not expect.
